# 箭头操作符

也就是`->` .

## 原生定义

**定义**：调用（指针的**解引用**）的成员。

​			针对类指针的小小语法糖。

**实现**：

```cpp
ClassName* e;
(*e).member = 1;
e->member = 1;
```

------

**妙用**：计算（类成员的）**偏移值**.

```cpp
int offset = (int)&((ClassName*)nullptr)->m_Member;
```

**用处**：当你把数据序列化为一串字节流时，用以计算某些东西的偏移量.
