# 处理多返回值

## std::tuple

**定义**：基本上是一个类，它可以包含多个变量而不关心类型.

**包含于库**：`<tuple>`

**实现**：

```cpp
std::tuple <ClassName/TypeName , ...> Var;//定义
```
